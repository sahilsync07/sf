# SF Srikakulam - Stock & Ledger Management System

Premium Stock Tracking and Ledger View for **Shree Footwear (SF Srikakulam)**.

## 🚀 Overview
SF Srikakulam is a high-performance, mobile-first web application designed for real-time stock monitoring and professional ledger management, integrated with Tally Prime.

## ✨ Key Features
- **Smart Catalog**: Instant brand-wise stock visibility with premium UI cards and detailed article info.
- **Tally Integration**: Dynamic data synchronization for both Stock and Ledgers via a secure API.
- **Professional PDF Generation**: Generate branded Purchase Orders and Ledger Reports with a single click.
- **Multi-Platform**: Native Android support via Capacitor and Offline-first PWA capabilities.
- **Admin Control**: Secure mode for image uploads and manual Tally syncs.

## 🛠️ Tech Stack
- **Frontend**: Vue.js 3, Vite, Pinia (State Management), Tailwind CSS (Aesthetics).
- **Backend**: Node.js, Express, Tally XML Interface.
- **Native**: Capacitor (iOS/Android).
- **Storage**: Cloudinary (Image Hosting), LocalStorage/Cache API (Offline Support).

## 📦 Setup & Development

### Local Development
1. Clone the repository.
2. Run `npm install` in both the root and `Frontend` directories.
3. Use the root helper to start both services:
   ```bash
   .\start-server.bat
   ```

### Application URLs
- **Frontend**: http://localhost:5173
- **Backend**: http://localhost:3000

## 📱 Mobile App
The project includes a ready-to-run Android application in the `Frontend/android` directory.
